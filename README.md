## How to use this Todo App
***
**After opening the application, you will see the start page.**

In the **top left corner** there will be a "burger button" in which you can set a name / nickname for the application.
####
At the **bottom right** there is a button to add a new category.
####
After adding and moving to the added category, you will be able to add/view/delete or sort your tasks (today/tomorrow/all)
